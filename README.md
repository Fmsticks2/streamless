# Streamless

An autonomous on-chain subscription protocol built on Massa using Autonomous Smart Contracts (ASC) and DeWeb for decentralized frontend hosting.

## Monorepo Layout

- `contracts/` — Massa Autonomous Smart Contract (ASC)
- `frontend/` — React + Vite + Tailwind app
- `docs/` — Technical documentation
- `scripts/` — Deployment and helper scripts

## Quickstart

1. Frontend
   - `cd frontend`
   - `npm install`
   - `npm run dev`
2. Contracts (AssemblyScript or Rust)
   - See `contracts/README.md`
